================================================================================
MANUAL INSTALLATION STEPS - Windows SDK
================================================================================

PROBLEM:
  better-sqlite3 requires Windows SDK to compile, but it's missing.

SOLUTION:
  Install Windows SDK through Visual Studio Installer.

================================================================================
STEP-BY-STEP INSTRUCTIONS:
================================================================================

STEP 1: Open Visual Studio Installer
  - Press Windows Key
  - Type "Visual Studio Installer"
  - Press Enter
  - OR download from: https://visualstudio.microsoft.com/downloads/

STEP 2: Modify Visual Studio 2019 Build Tools
  - Find "Visual Studio 2019 Build Tools" in the list
  - Click the "Modify" button (NOT "Launch")

STEP 3: Add Windows SDK Component
  - Click the "Individual components" tab (at the top of the dialog)
  - In the search box, type: "Windows SDK"
  - Look for and CHECK (select) one of these:
    * Windows 10 SDK (10.0.19041.0) or newer
    * Windows 11 SDK (10.0.22000.0) or newer
  - Make sure there's a checkmark next to it

STEP 4: Install
  - Click the "Modify" button at the bottom right
  - Wait for installation to complete (10-20 minutes)
  - You may be asked to restart your computer

STEP 5: Install Dependencies
  - Open PowerShell in the project directory
  - Run: npm install
  - This should now work without errors!

STEP 6: Run the Application
  - Run: npm run electron-dev
  - The app should start!

================================================================================
ALTERNATIVE: If Visual Studio Installer is Not Found
================================================================================

OPTION 1: Download Visual Studio Installer
  - Go to: https://visualstudio.microsoft.com/downloads/
  - Download "Build Tools for Visual Studio 2019"
  - Install it
  - Then follow steps above

OPTION 2: Install Visual Studio Community (Easier)
  - Go to: https://visualstudio.microsoft.com/vs/community/
  - Download and install Visual Studio Community
  - During installation, select "Desktop development with C++" workload
  - This automatically includes Windows SDK
  - Then run: npm install

================================================================================
VERIFICATION:
================================================================================

After installing Windows SDK, verify it worked:

1. Open Visual Studio Installer
2. Click "Modify" on Visual Studio 2019 Build Tools
3. Go to "Individual components" tab
4. Search for "Windows SDK"
5. You should see the Windows SDK you installed with a checkmark

Then run: npm install

If npm install succeeds, you're done!
